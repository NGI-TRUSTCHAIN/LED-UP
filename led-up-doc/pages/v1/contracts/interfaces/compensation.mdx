This interface defines the functions for handling compensation-related operations.

## Functions

### processPayment

Processes a payment for a given producer based on the data size.

```solidity
function processPayment(address _producer, string memory _recordId, uint256 dataSize) external;
```

**Parameters**

| Name        | Type      | Description                                                    |
| ----------- | --------- | -------------------------------------------------------------- |
| `_producer` | `address` | The address of the producer receiving the payment.             |
| `_recordId` | `string`  | The ID of the record associated with the payment.              |
| `dataSize`  | `uint256` | The size of the data for which the payment is being processed. |

### verifyPayment

Verifies whether a payment has been made for a specific record.

```solidity
function verifyPayment(string memory _recordId) external view returns (bool);
```

**Parameters**

| Name        | Type     | Description                                     |
| ----------- | -------- | ----------------------------------------------- |
| `_recordId` | `string` | The ID of the record to verify the payment for. |

**Returns**

| Name     | Type   | Description                                              |
| -------- | ------ | -------------------------------------------------------- |
| `<none>` | `bool` | True if the payment has been processed, otherwise false. |
